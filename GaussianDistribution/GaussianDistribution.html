一、创建正太分布¶

  • $N(0,1)$:np.random.randn 创建标准正太分布
  • $N(\mu,\delta)$:$x = \mu + \delta N(0,1)$
In [8]:
# 标准正太分布散布
x : np.ndarray = np.random.randn(1000,1)

# 标准正太分布N(0,1)转非标准N(miu,variance)
mu = 20
variance = 10
x : np.ndarray = mu + variance * x

二、计算数据的均值与方差¶

$$ \delta = \sqrt[]{\frac{(x_1 - \mu)^2 + \dotsm + (x_n - \mu)^2}{n} } $$
In [9]:
# 计算正太分布的均值，方差
mu = x.mean() 
variance = sqrt(np.sum((x-mu)**2,axis=0)[0] / len(x) )

三、正太分布概率函数¶

$$ f(x)=\frac{1}{\sqrt{2 \pi} \sigma} \exp \left(-\frac{(x-\mu)^{2}}{2 \sigma^{2}}\right) $$
In [10]:
# 计算正太分布的概率函数
fx = 1/sqrt(2 * pi) / variance * np.exp(-(x - mu)**2/2/variance**2)
